# Line PDF Bot Documentation

這是一個使用 LangChain、FastAPI 和 Vue 構建的 Line Bot，可以透過 Line Liff 網頁上傳 PDF 文件並回答相關問題。

## 功能
目前支援以下幾種功能:
- 上傳 PDF 文件
- 自然語言處理，從 PDF 中提取相關內容回答問題
- 自動 Embedding 文件
- 使用 Line Bot 進行互動
